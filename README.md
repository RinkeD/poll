### Use
1. Clone source
2. `npm run server`
3. Browse to `localhost:80`
4. Poll away!

* [Express](http://expressjs.com/)
* [ezlog](https://github.com/opensoars/ezlog)
* [cls](https://github.com/opensoars/cls)
* HTML5
